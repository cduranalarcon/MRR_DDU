Comprar WinRAR ahora

Como activar su licencia

Si usa WinRAR solo necesita abrir el fichero rarkey.rar que le enviamos con WinRAR y responder "Sí" a la pregunta

También puede copiar el fichero rarreg.key en la carpeta de WinRAR o en la carpeta %APPDATA%\WinRAR. Por defecto la carpeta de WinRAR es "C:\Program Files\WinRAR", pero la puede cambiar el usuario al instalar WinRAR. También puede arrastrar y soltar el fichero rarreg.key en una ventana de WinRAR para activar la licéncia.

En el caso de RAR para Unix o para OSX debe copiar el fichero rarreg.key en su directorio raíz o a uno de los siguientes direcotrios: /usr/lib, /usr/local/etc. Puede renombrarla a .rarreg.key o .rarregkey, si lo desea, aunque rarreg.key tambien es válido.

Ahora WinRAR, RAR para Unix y OS X usan el mismo formato de llave de registro, de forma que puede usar la misma llave con las versiones actuales de WinRAR y RAR para las plataformas mencionadas. No se garantiza que versiones anteriores de RAR y WinRAR a la incluida en esta distribución puedan usar el mismo formato de llave, por ejemplo, las versiones anteriores a la 2.60 usan diferentes tipos de llaves.
